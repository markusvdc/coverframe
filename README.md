## HAVE A'GANDER
Cover Frame is a lightweight Figma plugin that brings CSS-like object-fit: cover behavior to your designs. With a single click, it automatically resizes and centers images to fully cover their parent frames while preserving the original aspect ratio. The plugin also enables Clip Content to hide overflowing areas, creating a seamless and intuitive workflow. Built to solve a missing native feature in Figma, Cover Frame is perfect for wallpapers, Pinterest pins, thumbnails and any image-based layout that requires fast and precise cropping.
